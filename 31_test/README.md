## 为什么没人说【回归测试】了？

回归测试：
当修复一个BUG后，把之前的测试用例在新的代码下进行再次测试。 确保改完原来的BUG后，没有给其他部分带来新的缺陷。

随着自动化测试的发现，现代 CI 中都会把测试用例再次测试一遍，也就是说回归测试被自动化了。
